# Voice Assistant - Command Reference

Your personal Jarvis-style AI assistant for task management!

## 🎙️ How to Use

1. Click the **"Talk to Assistant"** button on your Home page
2. Wait for the microphone to activate
3. Speak clearly and naturally
4. The assistant will respond both visually and verbally

## 💬 Conversational Features

### Greetings & Interaction
- "Hi" / "Hello" / "Hey"
- "Good morning" / "Good afternoon"
- "Thank you"
- "Help" / "What can you do?"

### Status & Information
- **"What's on my agenda today?"** - Shows today's tasks
- **"What should I work on next?"** - Intelligent task recommendation
- **"How many tasks do I have?"** - Counts pending and completed tasks
- **"Show my projects"** - Lists all your projects
- **"List my tasks"** - Shows pending tasks

### Creating Tasks
**Basic:**
- "Add task [task name]"
- "Create task [task name]"

**With Date:**
- "Add task buy groceries tomorrow"
- "Add task client meeting on Monday"
- "Add task review code next week"

**With Time:**
- "Add task call mom at 3pm"
- "Add task dentist appointment tomorrow at 2:30pm"

**With Project:**
- "Add task design mockups to Website Redesign"
- "Add buy domain to Launch Project tomorrow"

**Full Examples:**
- "Add task buy groceries tomorrow at 3pm"
- "Create task finish report for Marketing on Friday at 9am"
- "Add prepare presentation to Q1 Planning next Monday"

### Creating Projects
- "Create project [project name]"
- "New project [project name]"

**Examples:**
- "Create project Website Redesign"
- "New project Q1 Marketing Campaign"

## 🧠 Smart Features

### Context Awareness
The assistant remembers:
- Your last created project
- Your last created task
- Conversation flow

### Natural Language Parsing
The assistant understands:
- **Relative dates**: today, tomorrow, next week
- **Days of week**: Monday, Tuesday, etc.
- **Times**: 3pm, 2:30pm, 9am
- **Project references**: "to [project]", "for [project]", "in [project]"

### Intelligent Recommendations
- Prioritizes high-priority tasks
- Considers due dates
- Suggests what to work on next
- Tracks task completion

## 🔊 Voice Responses

The assistant speaks back to you! You can:
- **Toggle voice on/off** using the speaker icon
- Hear confirmations when tasks are created
- Get verbal feedback on queries
- Receive error messages spoken aloud

## 💡 Pro Tips

1. **Speak clearly** - Pause slightly after clicking the microphone
2. **Be specific** - Include dates and times when scheduling
3. **Use project names** - Reference projects by their exact name
4. **Ask for help** - Say "help" anytime to hear what's possible
5. **Natural language** - No need for robotic speech, talk normally!

## 🐛 Troubleshooting

**"Voice assistant not supported"**
- Use Chrome, Edge, or Safari browser
- Ensure microphone permissions are granted

**Assistant doesn't hear you**
- Check microphone permissions in browser settings
- Ensure microphone is not muted
- Try speaking closer to the microphone

**Voice responses not working**
- Check that the speaker icon shows sound enabled
- Ensure system volume is up
- Try toggling voice off and on
